Project Overview

Bridging communication for the Deaf and hard-of-hearing community

SignSight is a research initiative at the Sri Lanka Institute of Information Technology that applies computer vision and deep recurrent networks to translate dynamic hand gestures into readable text in real time. The system is built around MediaPipe Holistic landmarks, StandardScaler and PCA preprocessing, and stacked Bi-LSTM / Bi-GRU temporal classifiers.

By relying only on a consumer-grade webcam and open-source tools — OpenCV, TensorFlow, MediaPipe, scikit-learn — the pipeline stays lightweight, reproducible, and honest about its limits, making it accessible for schools, NGOs, and community settings across Sri Lanka.

Key Features

What the system does

A complete, end-to-end recognition stack — from raw webcam frames to predicted glosses — with preprocessing, training, and inference aligned across the pipeline.

Real-Time Capture

Uses ordinary webcams with MediaPipe Holistic to extract 33 pose, 468 face, and 42 hand landmarks per frame — no depth sensors, no gloves.

Dynamic Sequence Model

A 3-layer bidirectional LSTM (64-128-64) with recurrent dropout captures temporal structure across a 45-frame window.

Lightweight Bi-GRU Baseline

A parallel Bi-GRU model with fewer parameters and a 32-frame buffer, offering a lighter deployment option for low-power devices.

Train-Serve Consistency

Scaler, PCA, and sequence length rules are serialized and shared between training notebooks and the live inference pipeline.

Confidence Scoring

Each prediction is returned with a 0–100 Recognition Confidence Score and optional SHAP-style feature attribution.

Accessibility-First UI

A cross-platform web/mobile client with large icons, high-contrast visuals, and intuitive feedback tailored for hearing-impaired users.

Technology Stack

Built with open, reproducible tools

The entire pipeline runs on a single consumer laptop, using libraries that students and small labs can install in minutes.

Research Highlights

What makes this work stand out

A focused, honest, deployment-oriented contribution to low-resource sign language recognition.

15-Class Tamil Sign Language Corpus

68 clips per gloss across everyday greetings, colours, and actions — 1,020 sequences in total, collected with consumer webcams.

Side-by-Side Architecture Comparison

Bi-LSTM (45-frame, 92.16%) vs Bi-GRU (32-frame, 93.46%) under identical preprocessing — documented in IEEE format.

Modular SignSight Integration

Clear interface contracts for sibling modules: audio-to-sign, TSL learning platform, and emotion-aware child support.

Community-Facing Design

Low hardware cost, transparent limitations, and a roadmap for signer-independent evaluation with Deaf-led organisations.
