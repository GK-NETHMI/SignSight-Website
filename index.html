Project Overview

Bridging communication for the Deaf and hard-of-hearing community. See the Sign. Learn the Language.

is a BSc (Hons) thesis project developed by a team of four Information Technology students under Project ID 25-26J-404. The platform is designed to bridge the communication gap faced by Tamil-speaking individuals with hearing impairments by providing a comprehensive Tamil Sign Language (TSL) support system. It combines machine learning, computer vision, and speech recognition to convert text, audio, and video content into TSL, while also offering an interactive learning and evaluation environment. The system further extends its reach to children through an emotionally aware module that monitors engagement and delivers insights to guardians.

Key Features

What the system does

A complete, end-to-end Tamil Sign Language platform — from audio/video input to sign output — covering learning, detection, emotion awareness, and accessibility.

Audio/Video to TSL

Converts spoken Tamil audio or uploaded video into Tamil Sign Language output using speech recognition and machine learning in near real-time.

Real-time gesture detection

Captures TSL gestures via webcam using MediaPipe Holistic — extracting pose, face, and hand landmarks per frame with no depth sensors or gloves required.

Learning & evaluation

An interactive platform with multi-level quizzes and video-based sign assessments that tracks student progress and provides performance insights for mentors.

LSTM sequence model

A temporal deep learning model captures gesture sequences across frames, enabling accurate dynamic sign recognition beyond static hand positions.

Emotion-aware children's mode

Detects facial emotions of children while watching content using MobileNetV2, then auto-generates guardian-friendly reports with learning tips via email.

Explainable AI insights

SHAP-based feature attribution gives transparent confidence scores for sign predictions, helping students and educators understand model decisions.

Research Highlights

What makes this work stand out

A focused, honest, deployment-oriented contribution to low-resource sign language recognition.

15-Class Tamil Sign Language Corpus

68 clips per gloss across everyday greetings, colours, and actions — 1,020 sequences in total, collected with consumer webcams.

Side-by-Side Architecture Comparison

Bi-LSTM (45-frame, 92.16%) vs Bi-GRU (32-frame, 93.46%) under identical preprocessing — documented in IEEE format.

Modular SignSight Integration

Clear interface contracts for sibling modules: audio-to-sign, TSL learning platform, and emotion-aware child support.

Community-Facing Design

Low hardware cost, transparent limitations, and a roadmap for signer-independent evaluation with Deaf-led organisations.
